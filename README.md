### Windows PowerShell "UnauthorizedAccess" Error
If you see an error like `File ...\npm.ps1 cannot be loaded`, this is a Windows security feature.

**Fix:** Run this command in your PowerShell window:
```powershell
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
```
Then try `npm run dev` again.

Alternatively, use **Command Prompt (cmd)** instead of PowerShell.
